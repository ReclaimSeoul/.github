# Reclaim Seoul

**Reclaim Seoul** is a digital repository and toolkit for exploring combinatorial material reuse in the context of Seoul’s temporary, transformed, and residual urban material streams.

The organization hosts shared datasets, workshop code, design tools, and student project repositories developed for **“Reclaim Seoul: Combinatorial Material Reuse for a Temporal City”**, part of **AAVS Seoul 2026**.

The work connects reclaimed materials, computational design, discrete aggregation, digital cataloguing, and assembly workflows using tools such as **Wasp**, web-based viewers, and experimental AR interfaces.

Official website: https://www.social-algorithms.com/whats-on/2026-unit-3

## Repositories

This organization may include:

* material datasets and metadata
* Wasp workflows and examples
* web-based design and visualization tools
* AR assembly prototypes
* student group repositories
* final project documentation

The goal is to build an open and collaborative workspace for documenting, testing, and sharing design methods for material reuse.
